Square a user-inputted integer

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a Java program that takes an integer from the user and squares it. This program should print the square of the input number.
Sample Input:
5

Sample Output:
25


Sample Input:
-3

Sample Output:
9



Memory limit: 256 MB
Time limit: 5 seconds
